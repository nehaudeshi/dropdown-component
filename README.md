# Instructions 

1. Download the project
2. Run `npm install` to install all the dependencies
3. Run `npm start` to run the app in development mode.
4. Use `DropdownComponent` to use the dropdown component.
5. For changing the label, use the `label` prop.
6. For multi select, change `isMulti` to true.
7. Keep your dropdown items in the `data` prop (as an array).


Checkout the website : [Dropdown Example](https://hiveai-dropdown.netlify.app/)
